What Are These Toolkits?

Instead of visiting separate websites for each small task, our toolkits group related tools into unified pages. Each toolkit is self-contained — download it as one HTML file and keep it on your computer for offline use.

How to Use Them

  1. Browse the six categories below and select the one matching your current project or task
  2. Click "Open toolkit" to launch all tools in that category at once
  3. Switch between tools within the toolkit using the built-in navigation tabs
  4. Save the page (Ctrl+S or Cmd+S) if you want permanent offline access

Technical Details

  • Each toolkit is a single HTML file with embedded CSS and JavaScript (no external dependencies after first load)
  • Data persists in your browser's localStorage between sessions (scores, notes, preferences)
  • No server-side processing occurs — calculations and generation happen on your device only
  • Compatible with Chrome, Firefox, Safari, Edge, and most mobile browsers
  • Total combined size of all six toolkits is under 500KB (lighter than most single web apps)

Frequently Asked Questions

Can I modify the toolkits or add my own tools?

Since each toolkit is a standard HTML file, you can open it in any text editor and customize the code, add new tools, or remove ones you don't need. The code is uncompressed and commented for readability.

Do the tools sync across devices?

Not automatically. Each device stores its own data locally. If you want synchronization, you could host the file on a private cloud storage service (like Dropbox or Google Drive) and access it from multiple browsers.

What's the difference between these toolkits and your individual tool pages?

The functionality is identical. Toolkits simply group related tools into a tabbed interface so you can switch between them without navigating to different URLs. Choose whichever interface you prefer — individual pages or grouped toolkits.
