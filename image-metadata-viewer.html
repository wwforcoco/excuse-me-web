What This Tool Does

Beyond the pixels you can see, an image file carries details such as its size, dimensions, and format. JPG files often also carry EXIF data written by a camera or phone, like the make, model, and date a photo was taken. This tool reads and lists whatever of that information is present in your file.

Steps to Check a File

  1. Load a picture into the box above
  2. Review the file details table that appears
  3. If the file is a JPG with EXIF data, camera details appear in a second table
  4. Load a different file at any time to check another picture

What Gets Checked

  • File name, type, size, and last modified date
  • Pixel width and height read directly from the image
  • EXIF fields such as camera make, model, and capture date when present in a JPG
  • A clear note when no EXIF data is found, rather than a blank screen
  • Everything is read locally, so private photo details are never sent anywhere

Frequently Asked Questions

Why do some pictures show no EXIF data?

EXIF is only embedded by certain devices and software, and many websites strip it out when a photo is uploaded or edited. PNG and WebP files typically don't carry EXIF at all.

Does this show my location if the photo has GPS data?

If GPS coordinates are embedded in the file, they will be listed. Be careful sharing such photos publicly if you'd rather not reveal where they were taken.

Can I remove EXIF data with this tool?

This tool only reads and displays metadata. To strip it, resave the picture through the resizer or compressor tool, which produces a fresh file without the original EXIF block.

Why is the file size shown here different from what my computer reports?

They should match exactly, since both read the same underlying file. A mismatch usually means a different file was loaded than expected.

Does this work for screenshots?

Yes, though screenshots rarely contain camera EXIF data since they aren't captured by a camera sensor.
